# TL;DR

## Require Node

- Node version 20

## Install PNPM

```
npm i -g pnpm
```

## Install Library

```
pnpm run install-library
```

# Development

```
pnpm run dev
```

# Production Build && Start Production

```
pnpm run build
pnpm run start
```

# Analyze Production Build Size

```
pnpm run analyze
```
